动漫主题－电子游戏主题－ACG专题－模板说明


《相聚一刻》又译《一刻馆》，是高桥留美子的漫画作品。该作于1980年在小学馆青年漫画杂志《Big Comic Spirits》创刊号上开始连载，后来改编成电视动画，全96话。
めぞん是法语Maison的音译，是“屋”或“馆”的意思。
《相聚一刻》主要在描述一刻馆的房客－大学落榜生五代裕作，和年轻守寡的管理员音无响子以及其他房客之间的互动。
《相聚一刻》动画版于86年三月开播后，外界反响一般，批评者多说动画版太着重《相聚一刻》的轻松一面，而忽略对角色感情的深入描写，演释手法亦较夸张，有失原作的平实风格。
或许是不如预期，当《相聚一刻》播至第27集，制作班底便来个大变身，剧本交付由《我系小奶油》而大放异彩的伊藤和典接手，至于人物设计方面则由森山雄治改由曾担任魔法小天使 、机动警察及橙路的高田明美(这也是后来有人常把《相聚一刻》中的音无响子 与《橙路》中的鲇川圆 相较的原因之一) ，这也是高田明美老师 为 高桥留美子老师作品动画化的第二弹，第一弹是福星小子。
除人物样貌行为有变，最令人惊喜是整套动画的风格趋向平实，由轻快谐趣转成着墨主角心路历程的变化，仔细又多元化的感情描述，从而成功转型而成为动画经典之作。

登场人物[编辑]
主要角色的姓氏中都含有数字：音无、一之濑、二阶堂、三鹰、四谷、五代、六本木、七尾、八神、九条、最后是响子的原姓－千草。
配音员（日本 / 香港TVB / 香港ATV/ 台湾）

五代裕作（配音员：二又一成 / 黄健强 / 翟耀辉 / 魏伯勤）

男主角，一刻馆五号房房客，个性优柔寡断，家人在故乡新潟县经营大众食堂。故事开始的时候，他是个准备重考的大学落榜生，后来考上了三流私立大学。在一刻馆的几年间，他不但遇到了理想伴侣，也在大学毕业后找到真正适合自己的职业──幼稚园老师，因此本作可说是他的成长故事。


音无响子（配音员：岛本须美 / 林元春 / 王梦华 / 李宛凤）

女主角，一刻馆的管理员，本姓千草，年轻貌美却是个寡妇。她不顾父母反对，与亡夫音无惣一郎结婚，展现出刚强个性。对爱情相当认真，对亡夫念念不忘，但在相处的过程中，不知不觉喜欢上五代，还会为他跟其他女性感情要好而吃醋。其“注册商标”是小鸡围裙与扫把。最后勇敢地接受五代的爱情与他结为连理，并生下一个健康的女婴──春香。


一之濑花枝 / 一之兰（配音员：青木和代 / 区瑞华）
一之濑先生 / 一之兰（配音员：矢田稔 / 张莺）
一之濑贤太郎（配音员：坂本千夏 / 蔡惠萍）

一之濑一家是住在一刻馆一号房房客。花枝可说是非常典型的欧巴桑，不但唯恐天下不乱，而且喜欢参加别人的聚会，还会大白天喝酒。贤太郎为其子，是个顽皮的小学生，喜捉弄五代。对母亲的行为相当反感，却又无能为力。花枝的丈夫在本作中虽有出场，但存在感近乎零，也没名字，因此一般都通称他一之濑氏（一之濑先生）。


二阶堂望（配音员：堀川亮 ※在原作与剧场版中登场，电视版无此角）

一刻馆二号房房客，大学生，家境殷实，起初搬来一刻馆时曾与其他房客格格不入，后来逐渐融入，但在事件上的反应可谓迟钝。


三鹰瞬（配音员：神谷明 / 林国雄 / 黄志成 / 官志宏）

响子的网球教练，家境优渥且体能卓越，特色是牙齿经常白得闪闪发光。热烈追求响子，多次向对方求婚但都被拒绝，是五代的情敌。唯一弱点是怕狗，一直设法克服。后来因误会而和九条明日菜结婚。


三越善三郎 （只在电视动画版出现）（配音员：堀胜之祐 / 冯志润）

一度住在一刻馆三号房的房客，在电视动画里只出现四集。


四谷（配音员：千叶繁 / 张炳强 / 梁志达）

一刻馆四号房房客，五代的邻居，也是个谜样角色，身份背景都是谜。曾用过“四谷菊千代”、“四谷剑之介”及“钱德勒四谷”等奇怪假名。平时多有骚扰五代的举动，常从墙洞偷窥五代生活。


六本木朱美（配音员：三田友子 / 姚天丽）

一刻馆六号房房客，在附近一家酒店“茶茶丸”当服务生。个性豪放泼辣，时常穿着内衣裤在公寓里晃荡，喜爱喝酒，也口无遮拦。


七尾梢/须惠/小津江（配音员：富永美衣奈 / 袁淑珍）

与五代在同一地方打工，也常邀五代回家吃饭，一直被当作五代的女友，而她似乎也不以为意，因此响子对她颇吃醋。大学毕业后，在银行里工作。因误会而和学长结婚，并定居名古屋。


八神吹雪（配音员：渊崎有里子 / 蔡惠萍）

五代在响子母校里当实习老师时的学生，家境敷裕，对五代抱有恋慕之心。知道响子和亡夫惣一郎是师生恋便效法她，对五代死缠烂打。


九条明日菜（配音员：鹤弘美 / 乐蔚）

旧华族千金，个性腼腆。与三鹰相亲认识。对他一见钟情，甚至得了相思病，不过跟三鹰相反的是──她非常喜欢狗。后来和三鹰结婚。


惣一郎（犬）（配音员：千叶繁 / 梅欣）

响子的爱犬，是亡夫惣一郎在路上捡回来杂种狗。因响子在呼唤其夫时，这只狗也连带有反应，因此后来取名叫惣一郎。
“惣”是“总”的异体字。


音无惣一郎（配音员：田中秀幸 / 林国雄）

响子的亡夫，也是她的高中老师。他与响子萌生师生恋，在响子父母反对下与响子结婚，但是婚后没多久就突然过世。由于故事一开始时他已过世，在回忆场面中都没出现真正长相。响子对他始终念念不忘，因此也成五代最大的情敌。


音无老人（配音员：槐柳二 / 谭炳文→源家祥）

音无惣一郎之父，一刻馆的主人。


音无郁子（配音员：庄真由美 / 黄丽芳 / 李宛凤）

音无惣一郎的侄女，曾聘请五代当家庭老师。


坂本（配音员：古川登志夫 / 冯志润）

五代的大学同学。


黑木小夜子（配音员：岛津冴子）

五代大学时同年级的社团同学，毕业后在幼稚园工作，曾请五代到幼稚园帮忙，让五代找到最适合自己走的路。


茶茶丸的老板（配音员：若本纪昭 / 谭炳文）

茶茶丸酒店老板，据说他的名字就是茶茶丸，后来与朱美结婚。


千草律子（配音员：松岛实 / 雷碧娜）

响子母亲，一直催响子脱离音无家的户籍并赶快再婚。最后发现五代是响子中意的对象。


响子之父（配音员：富田耕生 / 周鸾）

被响子妈妈压得死死的老爹，反对响子和惣一郎结婚，惣一郎死后亦反对响子再婚。


五代由加利（缘）（配音员：京田尚子 / 林丹凤）


五代的祖母，一直支持五代追求响子。


麦可罗（犬）

三鹰为了克服弱点而养的博美狗，和主人一样，牙齿都会闪闪发光。
